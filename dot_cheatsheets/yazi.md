# Yazi

## Navigation
| Key | Action |
|-----|--------|
| `h/j/k/l` | left/up/down/right |
| `Enter` | open file or directory |
| `q` | quit and cd to current directory |

## File Operations
| Key | Action |
|-----|--------|
| `y` | yank (copy) |
| `p` | paste |
| `d` | cut |
| `D` | delete |
| `r` | rename |
| `Space` | select |

## Finding
| Key | Action |
|-----|--------|
| `/` | search by name |
| `f` | filter files in current directory |
| `Z` | fuzzy jump via zoxide |

## Help
| Key | Action |
|-----|--------|
| `~` or `?` | open help menu |
